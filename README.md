## Deploy to GitHub Pages
1. Create a repository (e.g., `sayedrahim-portfolio`).
2. Add all files and folders.
3. Commit & push to GitHub.
4. In repository Settings -> Pages, select branch `main` and folder `/ (root)` and save.
5. Visit `https://<your-username>.github.io/<repo-name>/`

## Notes
- The contact form uses `mailto:` fallback (no backend). Replace with Formspree if needed.
- Update JSON files in `/data` to add projects and websites. They are fetched client-side.
- Make sure image files are placed in `images/`.
